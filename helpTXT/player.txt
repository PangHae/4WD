A description of the player.

There are four different states of hunger, thirst, behavior, and fatigue for the player.

All states have states from 0 to 10.

It is deducted from the activity of the next day.

If the hunger value is 0, the next day's fatigue is deducted by 1, and if the thirst is 0, the next day's fatigue is subtracted by 2.
If it is 5 or less, the next day's action is deducted by 5.
When the fatigue level reaches zero, it dies.
