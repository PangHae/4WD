게임을 플레이 하며 사용할 수 있는 command들은 다음과 같습니다.

gathering : 자원을 수집합니다.
-> -> 이 command들을 먼저 입력하고, 자원 이름을 입력합니다.
make : 아이템을 제작합니다.
-> 이 command를 먼저 입력하고, 만들고 싶은 아이템의 이름을 입력합니다.
use : 자원을 사용합니다.
-> 이 command들을 먼저 입력하고, 자원 이름을 입력합니다.
rescue : 배 또는 비행기를 통한 탈출을 시도합니다.
-> 이 command들을 입력하고, smoke 또는 sos를 입력합니다. 
-> smoke는 연기를 통해 주변을 지나는 배의 구조를 기다립니다.
-> sos는 주변을 지나는 비행기의 구조를 기다립니다.
sleep : 행동력을 모두 소모하여 다음 날로 날짜를 넘깁니다.
escape : 직접 제작한 배를 통한 탈출을 시도합니다.
save : 게임을 저장합니다. 입력하면 저장 파일 이름을 입력할 수 있습니다.
savelist : 저장파일 목록을 보여줍니다. 보여지는 형식은 ./Date/저장파일이름.txt입니다.
load : 저장된 데이터를 불러옵니다. 저장 파일 이름을 입력하면 불러올 수 있습니다.
