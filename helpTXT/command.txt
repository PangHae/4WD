게임을 플레이 하며 사용할 수 있는 command들은 다음과 같습니다.

gathering : 자원을 수집합니다.
make : 아이템을 제작합니다.
use : 자원을 사용합니다.
-> 이 command들을 먼저 입력하고, 자원이나 아이템의 이름을 입력합니다.

rescue (인자) : 배(smoke) 또는 비행기(sos)를 통한 탈출을 시도합니다.
sleep : 행동력을 모두 소모하여 다음 날로 날짜를 넘깁니다.
escape : 배를 통한 탈출을 시도합니다.
save : 파일을 세이브합니다.
savelist : 세이브 파일 목록을 보여줍니다.
load : 세이브 파일을 불러옵니다.
