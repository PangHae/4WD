command list

gathering : collect materials.
make : make items.
use : use materials.
->  example : command + parameter 

rescue (parameter) : try to rescue by ship(smoke) or plane(sos)
sleep : Use up all your action powers and pass the date to the next day.
escape : Try to escape through the ship.
save : save the file.
savelist : shows a list of save files.
load : load file.
